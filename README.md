# REACT CRA

This is a custom CLI for initializate a react app.

### Folders

```
├── app
│   ├── HelloWorld.tsx
│   ├── index.html
│   └── index.tsx
├── package-lock.json
├── package.json
├── tsconfig.json
└── webpack.config.js
```

### How to using?

Run this commands for create a react app

```
npm i -g @jmiranda1996/react-cra
react-cra my-app
cd my-app && npm start
```
